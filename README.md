This repository contains my coursework and homework assignments completed while learning **Angular** as part of my **Full Stack Developer Master Program**.

### 📚 About This Section

In this part of the program, I am learning how to build modern, scalable front-end applications using Angular. The exercises and assignments in this repository help me understand Angular’s architecture, component-based design, and best practices for building interactive web applications.

* Services and Dependency Injection
